Twitch.ME

Twitch.ME is a D3 based analytical website. The website is a twitch viewing dashboard based on data that is being pulled from the Twitch APIS. The Top 30 game table data is always live. You can also view The top 20 streamers for each game as well as watch those users stream by clicking on the barchart of that respective streamer.
